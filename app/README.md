# Pré-requisitos

- Node v18.20.3

# Instalação

UTILIZAR O YARN

```sh
$ yarn install
```

# Desenvolvimento

```sh
# Rodando a Aplicação
$ yarn start
```

ou o arquivo run.bat

```sh
# Rodando o Portal
$ yarn dev
```

Docker
Testar no docker antes de subir para os ambientes
